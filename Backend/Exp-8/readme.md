# Student Management REST API

A lightweight RESTful API built with Flask.

## Learning Outcomes

1. **Flask & Routing**: Mastered setting up a web server and defining endpoint routes.
2. **HTTP Methods**: Gained hands-on experience using `GET`, `POST`, `PUT`, and `DELETE`.
3. **JSON Data**: Learned to communicate between server and client using JSON.
4. **Code Structure**: Used Flask Blueprints to keep the project modular and organized.
5. **State Management**: Understood data flow and error handling using in-memory storage.
