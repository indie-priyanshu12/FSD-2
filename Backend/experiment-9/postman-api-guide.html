Simple Token Authentication

2 Steps
Step 1 — Get Token
Field Value
Method POST
URL http://localhost:5000/token-login
Body → raw → JSON
"username": "admin", "password": "admin123"
Response {"token": "YWRtaW4="} — Copy this token!
Step 2 — Use Token
Field Value
Method GET
URL http://localhost:5000/token-protected
Headers Key: x-auth-token  →  Value: (paste your token)
Expected Response 200 — "Token Auth Success. Welcome admin!"
4

JWT Authentication

2 Steps
Step 1 — Get JWT Token
Field Value
Method POST
URL http://localhost:5000/jwt-login
Body → raw → JSON
"username": "admin", "password": "admin123"
Response {"access_token": "eyJ0eX..."} — Copy this token!
Step 2 — Use JWT Token
Field Value
Method GET
URL http://localhost:5000/jwt-protected
Auth Tab → Type Bearer Token
Token (paste your access_token)
Expected Response 200 — "JWT Auth Success. Welcome admin!"
