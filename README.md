# FSD-2 Experiments Applications

This repository contains Full Stack Development (FSD-2) experiment applications for college coursework. The experiments cover a comprehensive range of topics from frontend development to backend services, testing, and deployment strategies.

## Experiment Overview

This collection of experiments is designed to provide hands-on experience with modern full-stack development practices and technologies.
